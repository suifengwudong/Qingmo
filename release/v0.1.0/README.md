# Release v0.1.0 - 轻墨小说写作工具

## 概述

轻墨是一款专为小说创作者打造的轻量化结构化写作工具。它复刻了VS Code的核心UI体验，剔除冗余功能，专注于小说写作的结构化需求。

## 新功能特性

### 🎯 核心功能
- **VS Code风格UI**：复刻左侧工具栏、树形导航、分栏编辑体验
- **模块化面板**：
  - 📝 小说编辑：Markdown编辑 + 项目树导航
  - 👤 人设&章节：图形化角色管理和章节时间轴
  - 🧭 大纲&伏笔：结构化大纲管理和伏笔追踪
  - 🤖 LLM辅助：智能写作辅助和对话优化

### 🎨 用户体验
- **轻量化设计**：安装包<5MB，启动<0.5秒，内存占用<50MB
- **中文支持**：内置Noto Sans CJK字体，完全支持中文显示
- **自动保存**：防止数据丢失，提升写作安全性
- **文件侧边栏**：默认隐藏，提升写作专注度

### 🔧 技术特性
- **本地优先**：纯文本存储（MD/JSON），无需云端同步
- **模块化架构**：清晰的代码结构，便于维护和扩展
- **跨平台支持**：基于Rust开发，支持Windows/macOS/Linux

## 系统要求

- **操作系统**：Windows 10+ / macOS 10.15+ / Ubuntu 18.04+
- **内存**：256MB RAM
- **存储**：50MB 可用空间

## 安装说明

1. 下载 `text_tool.exe` 可执行文件
2. 将文件放置在任意目录
3. 双击运行，无需安装

## 使用指南

### 快速开始
1. 启动应用程序
2. 点击"文件" → "打开项目"选择或创建小说项目目录
3. 使用左侧工具栏切换不同功能面板
4. 开始写作！

### 项目结构
小说项目将自动创建以下目录结构：
```
小说项目/
├── Content/     # Markdown内容文件
├── Design/      # JSON设计文件
└── 废稿/        # 废弃内容
```

### 核心操作
- **写作**：在小说编辑面板中编写Markdown内容
- **角色管理**：在人设面板中创建和编辑角色关系
- **结构规划**：在大纲面板中组织小说结构
- **智能辅助**：使用LLM面板获取写作建议
